# HAKODASE v2 オリジナリティ記録

## 目的
この文書は内部文書である。類似作品との関係、参考にした抽象的な仕組み、流用しないもの、HAKODASE独自の判断内容を記録する。

## 公開説明の方針
公開説明では既存ゲーム名を中心に置かない。HAKODASEは「箱を滑らせ、順番と倉庫設備を読み、同じ印の搬出口からすべて出荷する短時間パズル」として説明する。

## 参考にする抽象的な仕組み
- 盤面上の物体が止まるまで滑る。
- 同じ属性の出口から盤外へ出す。
- 出す順番を考える。
- 短時間のタイムアタックとして遊ぶ。

これらは抽象的な遊びの仕組みとして扱い、特定作品の名称、面データ、素材、文章、UI構成、演出をコピーしない。

## 流用しないもの
- 既存作品の名称。
- 既存作品の面データ。
- 既存作品の素材、画像、音声。
- 既存作品の文章、説明文、UI文言。
- 既存作品の画面構成。
- 既存作品の演出、広告表現、メタ進行。

## HAKODASE独自の判断内容
- 倉庫の出荷工程をテーマにする。
- 箱と搬出口に同じ色と記号を出し、色だけに頼らない。
- 複数の同色箱を、限られた搬出口から順番に出す。
- 出荷レーンと出荷シャッターを、倉庫設備として手順へ影響させる。
- 公式問題は検証済み問題集から選び、同じ問題の短いタイムアタックとして競う。
- Canvas2Dの軽量表現を基本とし、外部画像や外部音声なしで成立させる。

## 注意
既存ゲーム名は内部比較やリスク確認に限って扱う。公開文やストア説明では、HAKODASE自身の体験、テーマ、競技性を中心に説明する。
