## P2-02で実装したこと

### ホーム

- 一文説明
- 保存済みプレイヤー名
- 本日の出荷、エンドレス、練習
- スタート、遊び方、端末内記録、ゲームシェア、実験場導線

### 名前確認

```text
最大20文字
前後空白を除去
空文字不可
絵文字・記号を許可
確定後にblur()
端末内保存
```

保存領域が使えない場合も、有効な名前でそのプレイを開始できる。

### モード

- `daily`: 固定暫定seed。公式問題が未実装のため記録対象外と表示する。
- `endless`: seed付きランダム盤面。端末内記録だけ保存する。
- `practice`: 現行練習難易度と固定暫定seed。記録対象外とする。

### 端末内記録

- 新規記録へ`mode`を保存する。
- エンドレスだけ一覧へ表示する。
- モード情報を持たない旧v2記録は`legacy`とし、エンドレスへ混ぜない。

### 状態管理

- DOM画面を`AppController`へ接続した。
- 名前確定、再挑戦、新規盤面ごとに`RunController.prepare()`で新しい`playId`を発行する。
- P2-03までの暫定として、`countdown`状態を同期的に通過し、初回描画後にゲームとプレイ管理を開始する。

詳細は[P2-02設計](P2_02_HOME_NAME_MODES_v2.md)を参照する。
